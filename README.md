# Arkanoid

Proyecto desarrollado en Unity como práctica de sistema multimedia interactivo.

## Descripción

Este proyecto recrea el clásico juego Arkanoid utilizando Unity y C#.  
El jugador controla una pala que debe evitar que la pelota caiga mientras destruye bloques para conseguir puntuación.

## Controles

* Flecha izquierda → mover pala a la izquierda
* Flecha derecha → mover pala a la derecha
* Espacio → lanzar pelota
* Escape → pausa del juego

## Características actuales

* Sistema de puntuación
* Sistema de vidas
* Estados de juego
* Efectos visuales
* Sonidos
* Interfaz HUD
* Animaciones básicas

## Mejoras implementadas:



PaddleGrow — la pala se agranda temporalmente

PaddleShrink — la pala se encoge temporalmente

MultiBall — aparece una bola extra en juego

Los power-ups caen desde los bloques al destruirse con probabilidad configurable

Las bolas extra no restan vidas mientras haya más de una en juego

Los power-ups tienen animación en bucle mientras caen

Correción de aceleración de la bola

Cambio de color de los Sprites
